    Rocket Stabilization system

    FULL NAME

    RSS (Rocket Stabilization System; I am currently working on it)

    PROJECT DESCRIPTION

    There are a lot of variables in the rocket flight that cannot be predicted precisely. These are wind, humidity, temperature, etc. If you try to measure them beforehand, you would still be unable to predict their magnitude during the flight, or this measurement would not be inaccurate. Therefore, an active adjustment system is necessary to adapt the rocket's body parts (fins) to maintain stability and follow the intended trajectory. The main goal to be achieved is to quickly and precisely rotate the four fins with reference to the current position to maintain stability. This project faces a list of problems that need to be solved:

    1. The delay between getting measurements, calculating, and implementing them considering the high speed of the rocket. (The maximum delay must be 0.2 s between taking measures and finishing adjustment)
    2. The mass of the system needs to be considered due to the restrictions of The America Rocketry Challenge.
    3. The amount of pressure that each fin must withstand.
    4. Possible unpleasant weather conditions.

    I intend to use this technology in American Rocketry Challenge (2022).
